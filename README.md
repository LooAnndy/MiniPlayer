# MiniPlayer — 鸿蒙聚合音乐播放器

基于 **HarmonyOS NEXT API 23** + **ArkTS V2** 的多源音乐搜索与流媒体播放器。

---

## 功能实现

| 模块 | 状态 | 说明 |
|------|------|------|
| 网易云音乐 | ✅ | 搜索 + 流播放 |
| QQ 音乐 | ✅ | 搜索 + 流播放 |
| Bilibili | ✅ | 搜索 + 流播放 |
| Cookie 管理 | ✅ | 三平台独立配置 / 持久化 / 自动恢复 / 连接测试 |
| 多源切换 | ✅ | 搜索页内一键切换平台搜索源 |
| 播放控制 | ✅ | 播放 / 暂停 / 切歌 / 进度拖拽 / 播完重播 |
| 进度显示 | ✅ | 底部迷你播放条 + 全屏唱片播放页 |
| 本地文件 | ✅ | 选取本地音频 fd 播放 |
| 下载管理 | ⏳ | UI 已就绪，待接入下载服务 |

---

## 项目结构

```
entry/src/main/ets/
├── pages/          # 页面：启动/首页/搜索/播放/我的/下载
├── service/        # 服务：播放器引擎 + 三平台 API
├── view/           # 视图：播放条 + 歌曲列表项
├── viewmodel/      # 状态管理：播放器 ViewModel
├── model/          # 数据模型
└── utils/          # 工具：HTTP / 加密 / Cookie 存储 / 文件
```

---

## 快速开始

1. DevEco Studio + HarmonyOS SDK (API 23)
2. 打开项目，Build → Run
3. 进入「我的」→ 填写 Cookie → 测试连接
4. 搜索 → 播放
